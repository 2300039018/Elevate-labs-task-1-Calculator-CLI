# Task 1: Calculator CLI App

## Overview
This is a simple command-line calculator built in Python. It supports addition (+), subtraction (-), multiplication (*), and division (/), with error handling for division by zero. Built as part of Elevate Labs tasks.

## Features
- User-friendly menu for operations.
- Input validation for numbers and choices.
- Loops until user exits.
- Displays results with exact integer formatting where possible.

## How to Run
1. Ensure Python 3+ is installed.
2. Run: `python calculator.py`
3. Follow prompts: Choose operation (1-5), enter numbers.
